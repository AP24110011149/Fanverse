# FanVerse

FanVerse is a fan-inspired e-commerce platform for anime, movie, and pop-culture merchandise.

## Features
- User Authentication
- Product Catalog
- Shopping Cart
- Order Tracking
- User Profile Management
- Responsive UI Design

## Technologies Used
- HTML
- CSS
- JavaScript
- Firebase

## Project Structure
- Frontend Pages
- Product Management
- Cart System
- Authentication Module

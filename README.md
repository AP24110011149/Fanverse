# FanVerse 

FanVerse is a fan-inspired e-commerce platform for anime, movie, gaming, and pop-culture merchandise.

## Features

- User Authentication
- Product Catalog
- Shopping Cart
- Order Tracking
- User Profile Management
- Responsive Design
- Fan Edition Products
- Community Section

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase

## Project Highlights

- Modern responsive UI
- Product browsing and filtering
- Shopping cart functionality
- User account management
- Order tracking workflow
- Firebase integration
